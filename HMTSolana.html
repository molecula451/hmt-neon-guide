Prerequisites:

  • Metamask wallet setup
  • Add HMT to it: https://ropsten.etherscan.io/address/0xEedBf484B92858cB554a727Bff823b3C0a815f1c
  • Phantom wallet setup - Make sure you are on “Devnet”
  • Add this RPC to metamask:
  • Network Name: "remote proxy — solana devnet"
  • New RPC URL: https://proxy.devnet.neonlabs.org/solana
  • Chain ID: 245022926
  • Currency Symbol: NEON
  • Block Explorer URL : https://neonscan.org/
  • In this RPC make sure to have HMT (0x2a78ba72f52af2cc90c9389dbe0d2c4b10055b81) via "Import tokens"
  • Import HMT (0xEedBf484B92858cB554a727Bff823b3C0a815f1c) as a token in “Ropsten Test Network”
  • Some HMT and ETH in “Ropsten Test Network”

Steps:

  • Switch Metamask to “Ropsten Test Network”
  • Goto Portal Bridge UI
  • Select Ethereum (Ropsten) to Solana and click the “Connect” button
  • Metamask pops up and select your account
  • Goto Tab “Select a Token” and paste ”0xEedBf484B92858cB554a727Bff823b3C0a815f1c” there. You should be able to see HMT balance there.
  • Enter the amount in “Amount” tab and click “Next” and you will have to select Solana wallet - Select “Phantom” and something like this should be your screen
  • Click “CREATE ASSOCIATED TOKEN ACCOUNT” button (a token account in Solana is an account to receive the bridge tokens belonging to the Main Solana account)
  • Make sure you have some SOL balance you will need it for approvals, and to create the associated token account
  • Click “NEXT”
  • Click “TRANSFER”
  • Metamask pops up - “confirmations”; wait for it.
  • Click “REDEEM”
  • Approval screens pop up again and again - Give all approvals
    Final screen:
  • Check Phantom - It should be showing HMT tokens - Make sure you are in “Devnet” as shown in prerequisites

Now bridging HMT Tokens to NEON:

  • Goto bridge interface from Solana to Neon
  • Solana -> NEON
  • Select “Phantom” Wallet
  • In “Select a token” select “WHMT”
  • Should be showing the amount like shown below
  • Click on 0.0 and enter the amount you’d like to bridge to Neon
  • Connect the wallet to land -> Metamask pops up (make sure you are on solana/neon RPC)
  • Make sure you have some NEON and click “Next”

WHMT lands on Neon as a wrapped token you can follow the same steps to bridge back to first chain, at this point you may also use WHMT on this chain the way you’d like
